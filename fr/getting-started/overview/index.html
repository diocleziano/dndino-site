    Vue d'ensemble¶

    DnDino aide le maître du jeu à gérer :

    • les personnages
    • les aventures
    • les lieux
    • les présences dans les lieux
    • les combats
    • la session live pour les joueurs
    • le matériel de support

    Par où commencer¶

    Si tu veux prendre l'application en main sans te perdre, voici le parcours conseillé :

    1. lis la vue d'ensemble générale
    2. apprends à créer un personnage
    3. crée ta première aventure
    4. découvre la dashboard de l'aventure
    5. associe tes héros à l'aventure
    6. ajoute quelques images
    7. crée des lieux ou des quêtes dans l'aventure
    8. ajoute des PNJ ou des monstres à un lieu
    9. crée un combat de test avec les héros et les PNJ ou monstres de ce lieu
    10. fais un petit combat d'essai
    11. lorsque l'aventure grandit, utilise aussi les cartes conceptuelles et les cartes interactives pour mieux organiser la campagne

    Tip

    Il n'est pas nécessaire de tout configurer immédiatement. DnDino donne le meilleur de lui-même quand tu construis la campagne étape par étape : d'abord les personnages, puis l'aventure, ensuite les lieux et enfin le combat.
