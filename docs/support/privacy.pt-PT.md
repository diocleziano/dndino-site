# Privacidade

Ultima atualizacao: 9 de abril de 2026

DnDino e uma aplicacao macOS concebida para ajudar os Dungeon Masters a gerir aventuras, personagens, locais, combates, mapas e notas de sessao.

## Recolha de dados

DnDino nao recolhe dados pessoais para fins de publicidade, criacao de perfis ou rastreamento.

Os dados introduzidos na aplicacao, como aventuras, personagens, notas, imagens e conteudos relacionados, sao armazenados localmente no dispositivo do utilizador.

## Armazenamento local

A maior parte dos dados tratados pelo DnDino e armazenada localmente no Mac do utilizador, dentro do contentor de dados da aplicacao.

## iCloud

Se o utilizador optar por usar funcionalidades de sincronizacao ou snapshot baseadas em iCloud, alguns dados da aplicacao podem ser armazenados na conta iCloud do utilizador exclusivamente para permitir backup, sincronizacao ou restauracao de conteudos.

DnDino nao acede aos dados iCloud do utilizador para qualquer finalidade diferente da prestacao da funcionalidade pedida na aplicacao.

## Imagens e ficheiros escolhidos pelo utilizador

DnDino pode permitir ao utilizador selecionar imagens ou ficheiros do seu dispositivo. Esses ficheiros sao usados apenas dentro da aplicacao para as funcionalidades pedidas pelo utilizador.

## Rastreio e terceiros

DnDino nao utiliza sistemas de rastreamento publicitario e nao vende dados pessoais a terceiros.

## Seguranca

Os dados sao tratados usando os mecanismos de seguranca padrao fornecidos pelo macOS e pelas tecnologias Apple usadas pela aplicacao.
